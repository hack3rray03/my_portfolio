## This Repository

This repo contains the source code for my personal portfolio site — hand-coded in a single `index.html` file with no dependencies, no frameworks, and no build tools. Hosted via GitHub Pages.

**Design notes:**
- Dark terminal aesthetic with scan-line overlay and noise texture
- Monospace typography (`Share Tech Mono`) for the technical feel
- CSS custom properties throughout for easy theming
- Intersection Observer API for scroll-triggered animations
- Fully responsive — mobile down to 320px
